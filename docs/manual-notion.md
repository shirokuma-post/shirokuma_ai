# しろくまポスト ユーザーマニュアル

あなたの想いを、自動でSNSに届ける。AI SNS自動投稿サービス。

---

## しろくまポストとは？

しろくまポストは、あなたの想い・価値観・考えをベースに、AIがSNS投稿を自動生成・自動投稿してくれるサービスです。

APIキーを自分で持ち込む（BYOK: Bring Your Own Key）方式なので、月額料金を抑えながら好きなAIモデルを使えます。Threadsなら完全無料で始められます。

**対応SNS**: X（Twitter）/ Threads

**対応AIモデル**: Claude（Anthropic）/ GPT-4o（OpenAI）/ Gemini（Google）

## 初期セットアップ

### Step 1: SNSを選択

アカウント作成後、投稿先のSNSを選びます。

- **X（Twitter）**: X API Basic（$5/月）の契約が別途必要です
- **Threads**: Meta APIは無料。完全無料で始められます

> ⚠️ Free・Proプランでは、選択後にSNSの変更はできません。Businessプランでは両方利用可能です。

### Step 2: API接続（GPTsで簡単設定）

専用のGPTs（ChatGPT上のアシスタント）が対話形式でAPI接続をサポートします。

1. 「連携コードを発行」ボタンを押す
2. 6桁の連携コードが表示される（有効期限15分）
3. 「API接続GPTsを開く」ボタンでGPTsに移動
4. GPTsの対話に従い、APIキーの取得・入力を進める
5. GPTsが自動でしろくまポストにAPIキーを保存
6. チュートリアル画面が自動で次のステップに進む

**必要なAPIキー**:

| SNS | 必要なキー |
|-----|-----------|
| X | Consumer Key, Consumer Secret, Access Token, Access Token Secret |
| Threads | Access Token, User ID |

**AIのAPIキー（いずれか1つ）**:

| プロバイダー | キー形式 |
|------------|---------|
| Anthropic (Claude) | `sk-ant-...` |
| OpenAI (GPT) | `sk-...` |
| Google (Gemini) | `AIza...` |

### Step 3: あなたの想いを登録（マイコンセプト作成アシスタント）

専用のGPTs「マイコンセプト作成アシスタント」が、対話であなたの想い・価値観を引き出して構造化します。

1. 連携コードを発行
2. 「マイコンセプト作成アシスタントを開く」ボタンでGPTsに移動
3. 3つの質問に答える
    - Q1: あなたが伝えたいことは？（情熱の源泉）
    - Q2: それが届いた先でどんな変化が起きる？
    - Q3: その変化が広がった先に、どんな社会がある？
4. GPTsがAIで構造化し、しろくまポストに保存

構造化されたマイコンセプトは、すべての投稿生成のベースになります。

### Step 4: 完了！

セットアップ完了画面が表示されます。ここで、しろくまポストについてSNS投稿すると **Businessプランが3ヶ月無料** になるキャンペーンに参加できます。

## 投稿の生成と管理

### 投稿を生成する

「投稿」ページから、以下を選んでAIに投稿を生成させます。

**選択項目**:

1. **投稿先SNS** — X / Threads（Businessは両方選択可）
2. **スタイル** — 10種類から選択（プランにより制限あり）
3. **キャラ設定** — なし / 10種から選択（Pro以上）
4. **投稿の長さ** — Short / Standard / Long（プラン・SNSにより制限）
5. **分割モード** — フック＋リプライ形式（Business限定）

生成された投稿はプレビューで確認でき、編集・再生成・即時投稿・ドラフト保存が可能です。

### 分割投稿（Business限定）

1つの投稿を「フック（本文）」＋「リプライ（続き）」に分けて投稿します。フックで興味を引き、リプライで内容を展開するスレッド形式です。Xのスレッド文化と相性が良く、インプレッションが伸びやすいフォーマットです。

### 一括生成

「一括生成」ボタンで、スケジュールに設定されたすべてのスロット分のドラフトを一気に生成します。朝一括生成→確認→自動投稿の流れが基本です。

### 投稿ステータス

| scheduled | 投稿予定（時間待ち） |
| pending_approval | 承認待ち |
| posted | 投稿済み |
| failed | 投稿失敗 |

## スケジュール設定

毎日決まった時間に自動で投稿を生成・投稿できます。

### 基本設定

- **自動投稿の有効/無効**: マスタースイッチ
- **承認必須モード**: ONにすると、自動投稿前に承認が必要になります
- **トレンド注入**: ONにすると、RSSフィードから最新トピックを取り込んで投稿に反映します（Business限定）

### スケジュールスロット

各スロットで以下を設定できます:

- 投稿時間（HH:MM、日本時間）
- 投稿先SNS
- スタイル
- キャラ設定
- 投稿の長さ
- 分割モード

**スロット数の上限**:
Free = 3枠 / Pro = 10枠 / Business = 無制限

### 仕組み

1. 毎日深夜2時（JST）に全ユーザーのドラフトを一括生成
2. 各スロットの設定時間になったら自動投稿
3. 承認必須モードがONの場合、ドラフト生成後に通知→承認→投稿

## Learning機能（Pro以上）

過去にバズった投稿をAIに学習させ、今後の投稿品質を向上させる機能です。

### 使い方

1. 「Learning」ページを開く
2. 過去のバズった投稿をコピーして貼り付け
3. いいね数・インプレッション数を任意で入力
4. AIが自動で分析:
    - 構成パターン（導入→本文→締め）
    - フック（つかみ）の種類
    - トーン・口調
    - 効果があった手法
    - なぜバズったかの分析
5. 学習データは「AI最適化」スタイル使用時に自動で参照される

> 💡 3〜5投稿ほど登録すると効果が出始めます。いいね・インプレッションが多かった投稿を優先的に登録してください。

## よくある質問（Q&A）

### 基本

**Q: 完全に無料で使えますか？**
A: はい。Threadsを選択すれば、しろくまポスト自体の月額は¥0です。ただしAI APIの利用料（数円〜/投稿）はご自身のAPIアカウントに課金されます。Xの場合はX API Basic（$5/月）が別途必要です。

**Q: AIのAPI料金はどのくらいかかりますか？**
A: 1投稿あたりおおよそ1〜5円程度です（モデルや長さによります）。1日3投稿で月100円前後が目安です。

**Q: XとThreads、どちらがおすすめですか？**
A: コスト重視ならThreads（API無料）。既にXでフォロワーがいるならX。Businessプランなら両方使えます。

**Q: SNSの選択は後から変えられますか？**
A: Free・Proプランでは変更できません。Businessプランにアップグレードすると両方使えるようになります。

### セットアップ

**Q: GPTsとは何ですか？**
A: ChatGPT上で動くカスタムアシスタントです。しろくまポスト専用のGPTsが、対話形式でAPIキーの取得やマイコンセプトの構造化をサポートします。ChatGPTのアカウント（無料プランでOK）が必要です。

**Q: 連携コードの有効期限が切れました**
A: 「連携コードを発行」ボタンを再度押して、新しいコードを発行してください。有効期限は15分です。

**Q: GPTsでの設定がうまくいきません**
A: GPTsの指示に従って進めてください。それでもうまくいかない場合は、設定ページから手動でAPIキーを入力することもできます。

**Q: APIキーが正しいか確認できますか？**
A: 設定ページの「テスト生成」ボタンで確認できます。テスト投稿が生成されれば、APIキーは正常に動作しています。

### 投稿

**Q: 生成された投稿を編集できますか？**
A: はい。ドラフト状態の投稿はすべて編集可能です。内容を自由に書き換えてから投稿できます。

**Q: 投稿が気に入らない場合は？**
A: 「再生成」ボタンで新しい投稿を生成できます。スタイルやキャラを変えると違った切り口になります。

**Q: 同じような投稿ばかり生成されます**
A: 直近5投稿を自動で参照し、重複を避ける仕組みが入っています。スタイルを「ミックス」にすると、ランダムで様々なスタイルが使われます。

**Q: 投稿が失敗した場合は？**
A: SNS APIのエラーが原因です。設定ページでAPIキーが正しいか確認してください。X APIの場合、レートリミット（15分あたりの投稿制限）に引っかかっている可能性があります。

**Q: マイコンセプトを登録しなくても使えますか？**
A: 使えますが、AIは汎用的な内容を生成します。あなたの想いを登録すると、独自の切り口や主張が反映されるので、登録を強くおすすめします。

**Q: どのスタイルを使えばいいですか？**
A: まずは「ミックス」で1〜2週間回して、いいね・RTが多かったスタイルを確認してください。反応が良いスタイルが見つかったら、そのスタイルの比率を増やすのがおすすめです。

**Q: キャラ設定は必ず使うべきですか？**
A: いいえ。「なし」でも十分使えます。キャラは「いつもと違う切り口で書きたい」時の便利ツールです。堅い話をポップにしたいなら「ギャル」、知的に見せたいなら「哲学者」、のように場面で使い分けると効果的です。

### スケジュール

**Q: スケジュールの時間は日本時間ですか？**
A: はい。すべて日本時間（JST）で設定されます。

**Q: 承認必須モードとは？**
A: 自動投稿前にドラフトを確認・承認するフローです。ONにすると、生成されたドラフトは「承認待ち」になり、あなたが承認するまで投稿されません。

**Q: スケジュール投稿を一時停止したい場合は？**
A: スケジュールページの「自動投稿」トグルをOFFにすれば、すべての自動投稿が停止します。

**Q: 投稿時間のおすすめはありますか？**
A: 一般的にX/Threadsともに7〜8時（通勤時間）、12〜13時（昼休み）、20〜22時（夜のリラックスタイム）がエンゲージメントが高いと言われています。まずはこの3枠から始めてみてください。

### Learning

**Q: Learningにはどんな投稿を登録すればいいですか？**
A: いいね数やインプレッションが高かった投稿を登録してください。AIがその投稿の成功パターンを分析し、今後の生成に活かします。

**Q: Learning機能はすぐに効果が出ますか？**
A: 3〜5投稿ほど登録すると効果が出始めます。登録数が増えるほど、AIの分析精度が向上します。「AI最適化」スタイルを選ぶと、学習データが最大限活用されます。

### プラン・支払い

**Q: プランのアップグレード/ダウングレードはいつでもできますか？**
A: はい。いつでもプランページから変更可能です。Stripeの決済システムを使用しており、日割り計算されます。

**Q: 3ヶ月無料キャンペーンの後はどうなりますか？**
A: 期限の30日前からダッシュボードにお知らせが表示され、7日前と当日にメールでもお知らせします。期限切れ後は自動でFreeプランに戻ります。データは消えません。

**Q: 解約したらデータは消えますか？**
A: プランをダウングレードしてもデータは残ります。ただし、上位プラン限定の機能（Learning、分割投稿など）は使えなくなります。

### セキュリティ

**Q: APIキーは安全に保管されますか？**
A: すべてのAPIキーはAES-256-GCM方式で暗号化されてサーバーに保存されます。平文で保存されることはありません。

**Q: 他のユーザーから自分のデータが見えることはありますか？**
A: いいえ。Row Level Security（RLS）により、各ユーザーは自分のデータにのみアクセスできます。

**Q: APIキーを変更したい場合は？**
A: 設定ページからいつでも上書き保存できます。新しいキーを入力して保存すると、古いキーは上書きされます。

### 「投稿に失敗しました」と表示される
→ SNSのAPIキーが正しいか確認してください。Xの場合、APIプランがBasic以上であることを確認してください。Read-onlyアクセスでは投稿できません。

### 投稿が2重に投稿される
→ スケジュールと手動投稿が重複している可能性があります。スケジュール設定を確認してください。

### GPTsが「連携コードが無効です」と表示する
→ コードの有効期限（15分）が切れています。しろくまポストの画面に戻り、新しいコードを発行してください。
