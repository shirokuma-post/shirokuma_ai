# しろくまポスト 公式Threadsアカウント用マイコンセプト

## タイトル
しろくまポスト — 発信の継続を仕組みで解決するAIエンジン

## コンセプトテキスト

SNSで発信したい人は山ほどいる。自分の経験、知識、価値観、サービス。伝えたいことがある。でも、ほとんどの人が途中で止まる。

「何を書けばいいかわからない」
「書き始めても、まとまらない」
「忙しくて3日で途切れる」
「発信が大事なのはわかってる。でも日々の業務が先になる」
「投稿を書く時間で30分。それを毎日？無理」
「頑張って書いても反応ゼロ。モチベーションが消える」
「自分の文章に自信がない。誰かに添削してほしい」
「AIに書かせると、自分の言葉じゃなくなる気がして嫌」
「ネタが尽きる。1週間で書くことがなくなる」
「SNSのアルゴリズムとか、正直よくわからない」
「コンサルに頼むと月10万。そんな予算ない」
「ツール多すぎ。結局どれも続かなかった」
「そもそも、自分が何を伝えたいのか整理できてない」

これは才能の問題じゃない。意志の弱さでもない。「仕組み」がないだけ。

しろくまポストは、あなたの想い・価値観・経験をAIに預けて、毎日の投稿を自動で生成・配信する仕組み。

最初に一度だけ、あなたの想いを言語化する。「何を大事にしてるのか」「どんな変化を届けたいのか」「その先にどんな世界を見てるのか」。それをAIが理解して、あなたの言葉として毎日届けてくれる。

ただの文章自動生成ではない。10種類のスタイルで切り口を変える。共感、問いかけ、ストーリー、ぼやき、有益Tips。キャラクターで視点を変える。サラリーマンのリアルな共感、ギャルのポップな切り口、子どもの素直な疑問。時間帯でトーンを変える。朝は前向きに、昼は実用的に、夜は内省的に。直近の投稿と被らないように構造を毎回変える。同じ想いから、毎回違う形で届く。

続けていくうちに、反応が良かった投稿をAIに学習させられる。どんな構成が刺さったか、どんなフックが効いたか。AIがパターンを分析して、次の投稿に活かしてくれる。使えば使うほど、あなた専用の発信エンジンに育っていく。

しかも、APIキーを自分で持ち込む方式（BYOK）だから、月額課金のプレッシャーも少ない。AI利用料は1投稿あたり数円。Threadsなら完全無料で始められる。高額なSNSコンサルも、複雑なツールの使い分けも要らない。

セットアップも対話型。専用のGPTsアシスタントが質問に答えるだけで、APIキーの取得からあなたの想いの構造化まで全部やってくれる。技術的な知識は一切不要。

発信は「続けた人」が勝つ世界。でも続けること自体が一番難しい。だからこそ、続ける部分を仕組みに任せて、あなたは「何を伝えたいか」だけに集中すればいい。

「伝えたい」と思った瞬間に、もう発信は始まってる。あとは仕組みが届けてくれる。

しろくまポストは、発信を諦めかけた人のための、最後の仕組み。
